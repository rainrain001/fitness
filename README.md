# Fitness Food Tracker

**Know what you eat — and remember it.**

Fitness Food Tracker helps you save the foods you eat and keep track of their macronutrients: protein, carbohydrates, fat, and more. No more guessing or digging through old notes — build your own food library and see the nutrition breakdown at a glance.

## Why this project?

Whether you're meal prepping, hitting a protein goal, or just curious about what's on your plate, understanding macronutrients makes a real difference. This app makes it simple to:

- **Save foods** you eat regularly so you don't have to look them up every time
- **Track macronutrients** — protein, carbs, fat, and calories — for each item
- **Build a personal food library** that grows with your habits
- **Make informed choices** without the complexity of a full diet app

## How it works

1. Add a food you ate or plan to eat
2. Enter its macronutrient values (or pick from saved entries)
3. Your food library is ready whenever you need it

Small steps, clear numbers, less friction — so you can focus on your goals instead of spreadsheets.

## Setup

Make sure to install dependencies:

```bash
# npm
npm install
```

## Development Server

Start the development server on `http://localhost:3000`:

```bash
# npm
npm run dev
```

## Production

Build the application for production:

```bash
# npm
npm run build
```

Locally preview production build:

```bash
# npm
npm run preview
```

Check out the [deployment documentation](https://nuxt.com/docs/getting-started/deployment) for more information.
